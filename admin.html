¿Para qué sirve esto?
Con el token configurado, al hacer clic en "Publicar" el catálogo se actualiza automáticamente en el sitio — sin tocar código.

Cómo obtener el token (5 minutos):
1. Andá a github.com/settings/tokens/new
2. En "Note": personalizate admin
3. En "Expiration": elegí No expiration
4. Marcá solo el grupo repo (todos los permisos de repo)
5. Clic "Generate token" → copiá el token que aparece
6. Pegalo abajo y guardá

Para dar acceso a otra persona: simplemente compartís el link del admin y la contraseña. Cada uno configura su propio token de GitHub en su navegador.
