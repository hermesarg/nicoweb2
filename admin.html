⚠️ Cómo funciona este admin:
Este panel te permite preparar y gestionar las imágenes del catálogo localmente. Cuando agregás o eliminás una imagen, se genera el código HTML listo para copiar. Después solo tenés que:
1. Subir las imágenes nuevas a GitHub (Add file → Upload files)
2. Ir a la pestaña Exportar y copiar el HTML generado al index.html

💡 Tip: Guardá este archivo admin.html en tu computadora para usarlo siempre.

Pasos para publicar cambios:
1. Copiá el código de abajo
2. Andá a github.com/hermesarg/nicoweb2 → clic en index.html → ícono lápiz ✏️
3. Buscá la sección del catálogo (Ctrl+F: "cat-item") y reemplazala con este código
4. Commit changes → el sitio se actualiza en 2 min
5. Subí también las imágenes nuevas: Add file → Upload files
